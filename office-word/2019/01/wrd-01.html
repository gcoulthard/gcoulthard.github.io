Word 01: Getting Started

In this module, you are introduced to the basic features of Microsoft Word. You learn how to navigate and customize the Word application window, access menu commands from the Ribbon, review document template options, create and save documents, and perform basic data entry and editing. By the end of this module, you will be able to perform basic word processing tasks using Microsoft Word.

To follow along with the video lessons in this module, make sure
to download and unzip the Word 01 data files to your computer.

01. Introducing Word

By the end of this lesson, you will be able to:

  • Start and exit Microsoft Word

  • Identify and customize elements in Word’s application window

  • Adjust the document view and zoom options

  • Display and customize the Status bar options

  • Access the File menu for managing and printing files

  • View and specify document properties

Touring and Customizing Word (10:14m) — view on YouTube.

02. Templates & Editing

By the end of this lesson, you will be able to:

  • Navigate and search for Word document templates

  • Create, save, and close a document based on a Word template

  • Create a blank document and enter and delete text

  • Select text in a document

  • Use the Clipboard to cut, copy, and paste text

Creating and Editing a Document (10:48m) — view on YouTube.

03. Proofing Your Work

By the end of this lesson, you will be able to:

  • Use the Find command to find text within a large document

  • Use the Replace command to change text throughout a document

  • Run the Spelling Checker to check for spelling mistakes

  • Access the Thesaurus to find a word that better communicates your intent

Using Find, Replace, and Spell-Check (6:45m) — view on YouTube.
