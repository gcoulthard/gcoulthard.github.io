Word 03: Performing a Merge

A mail merge operation involves merging together a standard document, sometimes called a form-letter, with a dynamic content list, such as customer addresses and contact information. The result is the efficient production of personalized letters for hundreds or even thousands of individual people within minutes. In this module, you learn how to use Word to perform a mail merge operation. After creating a data source, you will prepare a form-letter template and then merge the source data into the template document. By the end of this lesson, you will possess the skills necessary to work on a direct mail campaign using Word.

To follow along with the video lessons in this module, make sure
to download and unzip the Word 03 data files to your computer.

01. The Mail Merge Wizard

By the end of this lesson, you will be able to:

  • Describe the elements required to perform a mail merge operation

  • Use the Mail Merge wizard to step through a mail merge

Using the Mail Merge Wizard (9:53m) — view on YouTube.

02. Merge Data Sources

By the end of this lesson, you will be able to:

  • Select and/or create a list to use for a data source

  • Sort and filter the records stored in a data source

  • Plan the structure of a data source using Word tables and Excel workbooks

Preparing the Data Source (6:28m) — view on YouTube.

03. A Custom Mail Merge

By the end of this lesson, you will be able to:

  • Create a form-letter template that may be re-used in a mail merge

  • Insert pre-defined merge blocks and fields into a form letter template

  • Customize the field display of a merge code

  • Preview the resulting merge operation

  • Save the results of a merge operation to an individual document

Performing a Custom Mail Merge (8:13m) — view on YouTube.
