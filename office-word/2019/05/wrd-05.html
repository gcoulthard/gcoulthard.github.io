Word 05: Structuring Your Document

There are several methods for customizing the display and presentation of a document. Rather than listing paragraphs of text, you will learn how to add a cover page, page numbering, and headers and footers to provide information and to lend structure to a document. Furthermore, you learn how to create document sections, which you may then use to change the number of columns displayed or to change document formatting options such as page orientation. By the end of this module, you will possess the skills necessary to create a well-structured and professional-looking document.

To follow along with the video lessons in this module, make sure
to download and unzip the Word 05 data files to your computer.

01. Cover & Page Numbering

By the end of this lesson, you will be able to:

  • Add a cover page and enter data into placeholder fields

  • Insert pre-formatted page numbers into a document

  • Specify whether to include page numbers on the first page

  • Position page numbers in the header, footer, or margins

  • Modify the number formatting and starting page number

Adding a Cover Page and Page Numbering (10:14m) — view on YouTube.

02. Headers & Footers

By the end of this lesson, you will be able to:

  • Add pre-formatted headers and footers to a document

  • Insert and edit content in a header and footer

  • Customize the appearance of a header and footer

  • Add a graphic logo, text box, or table to a header and footer

  • Modify the display options for a header and footer

Creating Headers and Footers (9:39m) — view on YouTube.

03. Sections & Columns

By the end of this lesson, you will be able to:

  • Choose paragraph formatting options for line and page breaks
