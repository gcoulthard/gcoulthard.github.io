Word 02: Enhancing Your Document

In this module, you learn how to apply character, paragraph, and document formatting commands in order to enhance the appearance of a document. You also learn how to use styles to ensure a consistent "look and feel." Besides working with text, you will insert, format, and position pictures and graphics to help communicate your message and to add clarity to your ideas. Lastly, you learn how to specify print layout options and send a document to the printer. By the end of this module, you will be familiar with Word’s formatting options for enhancing a variety of document types.

To follow along with the video lessons in this module, make sure
to download and unzip the Word 02 data files to your computer.

01. Formatting a Document

By the end of this lesson, you will be able to:

  • Apply character formatting commands, such as boldface, italic, and text colors

  • Apply paragraph formatting commands, such as lists, text alignment, and line spacing

  • Apply paragraph fill colors, borders, and shading options

  • Modify indent options and paragraph spacing

  • Use tabs, dot leaders, and line breaks in a document

Character and Paragraph Formatting (12:28m) — view on YouTube.

02. Styles & Themes

By the end of this lesson, you will be able to:

  • Select and apply professionally-designed styles to text selections

  • Modify the basic parameters of an existing style

  • Create your own formatting styles

  • Select and apply professionally-designed themes to a document

  • Modify the color, fonts, and effects used in a theme

Applying Styles and Themes (8:59m) — view on YouTube.

03. Text & Graphic Objects

By the end of this lesson, you will be able to:

  • Apply and modify special text effects

  • Insert and customize WordArt

  • Insert, position, and size a picture or graphic image

  • Adjust text wrapping and layout options for a picture

  • Apply a style and adjust the appearance of a picture

  • Insert, format, and add text to a shape object

  • Add a DropCap to the beginning of a paragraph

Working with Text and Graphics (7:29m) — view on YouTube.

04. Layout & Print Options

By the end of this lesson, you will be able to:

  • Add and remove a watermark in a document

  • Add a custom page color and border to a document

  • Adjust the margin settings for a printed page

  • Switch the page orientation between landscape and portrait

  • Select print settings to specify how a document should print

  • Preview and print a document

Page Layout and Print Options (4:21m) — view on YouTube.
