Excel 11: Functions and Macros

In this module, you learn more Excel functions for processing dates, text, and financial data. You are also introduced to macros for automating repetitive spreadsheet work and learn how to record, edit, and execute your own macros. By the end of this module, you will have the skills required to work more productively using Excelâ€™s more advanced features.

To follow along with the video lessons in this module, make sure
to download and unzip the Excel 11-12 data files to your computer.

01. Functions & Macros

By the end of this lesson, you will be able to:

  • Use the NOW and TODAY function

  • Use the ABS, RAND, and ROUND functions

  • Use the LOWER, UPPER, PROPER, and CONCATENATE functions

  • Use the PV and FV functions

  • Record a simple macro to be used repeatedly

  • Play back a macro using the keyboard and View Macros dialog box

  • Assign a macro to a button on a worksheet for execution
