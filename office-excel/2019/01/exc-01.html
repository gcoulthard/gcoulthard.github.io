Excel 01: Getting Started

In this module, you are introduced to the basic features of Microsoft Excel. You learn how to navigate and customize the Excel application window, access menu commands from the Ribbon, review worksheet template options, create and save workbooks, and perform basic data entry, editing, and formatting commands. By the end of this module, you will be able to perform basic spreadsheet tasks.

To follow along with the video lessons in this module, make sure
to download and unzip the Excel 01 data files to your computer.

01. Introducing Excel

By the end of this lesson, you will be able to:

  • Start Excel and open an existing workbook

  • Identify the primary elements in Excel’s application window

  • Adjust the worksheet view and zoom options

  • Navigate and rename sheet tabs

  • Format the tab color of sheet tabs

  • Insert, copy, and move sheets in a workbook

Touring Excel and the Workbook (11:11m) — view on YouTube.

02. Creating & Editing a Worksheet

By the end of this lesson, you will be able to:

  • Navigate and search for Excel workbook templates

  • Create a workbook based on an Excel template

  • Create a blank workbook and enter text, numbers, and dates

  • Select, edit, insert, and delete data in cells

  • Copy, move, and fill data in cells

  • Apply and clear number and date formats

Creating and Editing a Worksheet (13:45m) — view on YouTube.

03. Formatting & Printing a Worksheet

By the end of this lesson, you will be able to:

  • Change columns widths and row heights

  • Insert, delete, and hide columns and rows

  • Apply font and alignment formatting to cells

  • Merge and rotate cells

  • Add borders and fill color shading to cells

  • Set the print area and page layout options for printing

  • Preview and print a worksheet

Formatting and Printing a Worksheet (12:44m) — view on YouTube.
