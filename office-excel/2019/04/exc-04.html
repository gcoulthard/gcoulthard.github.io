Excel 04: Datasets and Tables

In this module, you are introduced to using Excel for managing lists of data. Rather than using a database program, such as Microsoft Access, you can store and manipulate thousands of rows of data in Excel with little impact on performance. In addition to inserting and deleting data, you learn to create and format tables, sort the contents into ascending and descending order, and then filter the table to display only the data you want to view.

To follow along with the video lessons in this module, make sure
to download and unzip the Excel 04 data files to your computer.

01. Sorting & Filtering Data

By the end of this lesson, you will be able to:

  • Sort a cell range or worksheet list by a single column of data

  • Perform a multi-column sort operation using the Sort dialog box

  • Display AutoFilter buttons in the header row of a worksheet list

  • Filter the display of data by numeric and text value

  • Clear individual filters or all filters from a worksheet

Sorting and Filtering Data (10:02m) — view on YouTube.

02. Working with Tables

By the end of this lesson, you will be able to:

  • Create a table using the Quick Analysis tool

  • Create a table using the Table command on the Ribbon

  • Format a table using Table Styles and options

  • Insert and delete rows and columns of data

  • Construct a structured reference formula in a table

  • Highlight cell values using conditional formatting options

  • Clear conditional formatting from cells, columns, or the worksheet

Creating and Formatting a Table (15:26m) — view on YouTube.
