Excel 07: Specialized Functions

In this module, you are introduced to using specialized Excel functions from the Date, Logical, and Financial function categories. You learn to use date functions for performing date math (e.g., how many days are there between two dates), logical functions for calculating statistics based on other values in the worksheet, and financial functions for preparing a loan amortization schedule.

To follow along with the video lessons in this module, make sure
to download and unzip the Excel 07 data files to your computer.

01. Date Functions

By the end of this lesson, you will be able to:

  • Use the NOW and TODAY functions to keep dates current

  • Use the YEAR, MONTH, DAY, and DATE functions to perform date math

  • Use the WEEKDAY function, along with VLOOKUP, to find the day of the week

Using Date Functions (8:34m) — view on YouTube.

02. Conditional Functions

By the end of this lesson, you will be able to:

  • Use the AND and OR functions for nested calculations

  • Use SUMIF to conditionally sum a range of values

  • Use AVERAGEIF to conditionally average a range of values

  • Use COUNTIF to conditionally count a range of values

Using Logical and Conditional Functions (7:31m) — view on YouTube.

03. Financial Functions

By the end of this lesson, you will be able to:

  • Use the iPMT and pPMT functions to calculate the periodic interest and principal paid on a loan

  • Use the CUMIPMT and CUMPRINC to calcalute the total interest and principal paid on a loan

  • Create a loan amortization table using Excel’s financial functions, including PMT

Creating a Loan Amortization Table (11:22m) — view on YouTube.
