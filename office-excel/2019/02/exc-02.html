Excel 02: Formulas and Functions

One of the primary strengths of Excel is its ability to quickly calculate mathematical equations based on worksheet data. In this module, you learn to use Excel’s most common built-in functions for summing, averaging, and analyzing the numeric data in a worksheet. You also learn to construct formulas using relative and absolute cell references, and how to use special functions to enable more complex calculations and data processing operations.

To follow along with the video lessons in this module, make sure
to download and unzip the Excel 02 data files to your computer.

01. Entering Quick Totals & Formulas

By the end of this lesson, you will be able to:

  • Use Quick Totals to insert calculations

  • Enter mathematical formulas into a cell

  • Display and hide cell formulas

  • Define named cell ranges from worksheet data

Entering Quick Totals and Formulas (11:19m) — view on YouTube.

02. Relative & Absolute Addresses

By the end of this lesson, you will be able to:

  • Describe the difference between relative and absolute cell addresses

  • Use both relative and absolute cell addresses in formulas

Using Relative and Absolute Cell References (6:39m) — view on YouTube.

03. Entering Common Functions

By the end of this lesson, you will be able to:

  • Use the SUM function to add together a cell range

  • Use the AVERAGE and MEDIAN functions to calculate the averages and mid-points

  • Use the COUNT and COUNTA functions to count the cells containing values

  • Use the MAX function to return the largest value in a cell range

  • Use the MIN function to return the lowest value in a cell range

SUM, AVERAGE, MEDIAN, MAX, MIN, COUNT, and COUNTA (9:06m) — view on YouTube.

04. More Advanced Functions

By the end of this lesson, you will be able to:

  • Use the VLOOKUP function to retrieve data from a cell range

  • Use the IFERROR function to test for calculation errors

  • Use the IF statement to make decisions within calculations

  • Use the PMT function to return the payment amount of a loan

VLOOKUP, IFERROR, IF, and PMT Functions (8:22m) — view on YouTube.
