Excel 03: Sparklines and Charts

Besides storing rows and columns of numeric values, Excel allows you to visually display data using charts or graphs. In this module, you learn to create sparklines (which are tiny charts appearing within cells), embedded charts (which are charts that float over top of the worksheet), and chart sheets based on worksheet data. Additionally, you learn to customize and format the appearance of chart data using titles, legends, colors, and graphics. These skills will help you convert simple text and numbers into beautiful charts that better communicate trends and patterns in the data.

To follow along with the video lessons in this module, make sure
to download and unzip the Excel 03 data files to your computer.

01. Using Sparklines

By the end of this lesson, you will be able to:

  • Plot cell data by adding a sparkline to a cell or cell range

  • Customize the appearance and colors of a sparkline

  • Display and hide markers in a sparkline

Adding and Customizing Sparklines (6:39m) — view on YouTube.

02. Working with Charts

By the end of this lesson, you will be able to:

  • Create an embedded chart using the Quick Analysis tool

  • Use the Recommend Charts feature to select new chart types

  • Size and move an embedded chart on a worksheet

  • Move an embedded chart to its own sheet tab

  • Switch the row and column data to plot in the chart

  • Remove and add data series to a chart

Creating and Modifying a Chart (8:58m) — view on YouTube.

03. Formatting a Chart

By the end of this lesson, you will be able to:

  • Add chart elements, such as axes, titles, data labels, and a legend

  • Adjust the appearance of chart elements using chart styles and colors

  • Customize the appearance of chart elements

  • Filter the display of items plotted in a chart

  • Add text boxes, shapes, and graphics to a chart

  • Set print options and send a chart to the printer

Formatting and Printing a Chart (15:04m) — view on YouTube.
