PowerPoint 04: Shapes, Animation, & Transitions

When used in moderation, object animation and slide transitions can add that "wow-factor" you need to engage an audience. However, when used too much, these features may detract from the message you are trying to communicate. In this module, you learn how to tastefully animate text, shapes, and other graphic objects, specify slide transitions, and ready a presentation for delivery.

To follow along with the video lessons in this module, make sure
to download and unzip the PowerPoint 04 data files to your computer.

01. Drawing & Formatting Shapes

By the end of this lesson, you will be able to:

  • Select and insert drawing shapes on a slide

  • Size, position, and copy drawing shapes

  • Insert and edit a free-form drawing shape

  • Format a shape using styles, fill colors, outline borders, and effects

  • Insert an action button on a slide

  • Create a hyperlink to a URL or to another slide in the presentation

Drawing and Formatting Shapes (12:48m) — view on YouTube.

02. Animating Text & Graphics

By the end of this lesson, you will be able to:

  • Add entrance, emphasis, and exit animation to slide objects

  • Modify the selected animation for an object

  • Use the Animation Pane to view and order animation effects

  • Specify advanced effect options for an animation

  • Create a custom motion path for animating an object

Animating Text and Graphics (9:58m) — view on YouTube.

03. Setting Slide Transitions & Timing

By the end of this lesson, you will be able to:

  • Add and preview a slide transition

  • Change the duration of a transition

  • Add sound and effects to a transition

  • Create a self-running slide show using timing options

  • Share and export a presentation by creating an MP4 video file

Setting Slide Transitions and Timings (8:11m) — view on YouTube.
