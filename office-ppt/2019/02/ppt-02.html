PowerPoint 02: Designing a Presentation

In this module, you learn to format the content placeholders in a slide presentation. You also insert and format text boxes and pictures in order to customize the placement and location of content on a slide. Lastly, you learn to use the Slide Masters in order to make changes to the entire presentation in one place. These skills will make you more efficient and productive in adding content and preparing slides.

To follow along with the video lessons in this module, make sure
to download and unzip the PowerPoint 02 data files to your computer.

01. Editing & Formatting Content

By the end of this lesson, you will be able to:

  • Create a presentation using the Blank template

  • Browse and apply design themes and variants

  • Add new slides with selected layouts

  • Edit and format text content within placeholders

  • Format selected placeholders using font, paragraph, and shape commands

Editing and Formatting Content (9:41m) — view on YouTube.

02. Working with Text & Pictures

By the end of this lesson, you will be able to:

  • Modify a slide layout by moving and sizing placeholders

  • Insert and format a text box on a slide

  • Insert and format WordArt on a slide

  • Insert and format pictures and photographs

Working with Text and Pictures (13:23m) — view on YouTube.

03. Using the Slide Master

By the end of this lesson, you will be able to:

  • Format the background of a slide

  • Apply a theme to selected slides only

  • View and modify the Slide Master placeholders

  • Insert new content placeholders on a slide

  • Modify a specific slide layout master

Using the Slide Master (8:23m) — view on YouTube.
