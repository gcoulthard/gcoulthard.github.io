PowerPoint 01: Getting Started

In this module, you are introduced to the basic features of Microsoft PowerPoint. You learn how to navigate and customize the PowerPoint application window, access menu commands from the Ribbon, review presentation template options, create and edit presentations, and annotate a slide presentation. By the end of this module, you will be able to create, view, and print a slide presentation using Microsoft PowerPoint.

To follow along with the video lessons in this module, make sure
to download and unzip the PowerPoint 01 data files to your computer.

01. Touring & Customizing PowerPoint

By the end of this lesson, you will be able to:

  • Start PowerPoint and open an existing presentation

  • Identify the primary elements in PowerPoint’s application window

  • Adjust the presentation view and zoom options

  • Display and reorder slides in Slide Sorter view

  • Display and reorder slides and content bullets on Outline view

  • Show the Ruler, Gridlines, and Guides in Normal Slide view

Touring and Customizing PowerPoint (12:28m) — view on YouTube.

02. Creating & Editing a Presentation

By the end of this lesson, you will be able to:

  • Navigate and search for PowerPoint presentation templates

  • Create a presentation based on a PowerPoint template

  • Enter and edit data in text placeholders

  • Add slide content using Outline view and Normal view

  • Insert a new slide and select a slide layout

  • Insert slides from a Word outline document

  • Reuse slides from an existing PowerPoint presentation

Creating and Editing a Presentation (12:28m) — view on YouTube.

03. Viewing & Printing a Presentation

By the end of this lesson, you will be able to:

  • Start and run a slide show from the beginning

  • Navigate to a specific slide in a slide show

  • Annotate a slide using Presenter tools

  • Use Presenter view to control a slide show

  • Specify print options, including headers and footers

  • Print a slide presentation and slide handouts

Viewing and Printing a Presentation (11:54m) — view on YouTube.
